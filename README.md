# React To Do List

React project to learn some React fundamentals such as components, state, props, JSX, events, etc.

[Tutorial](https://youtu.be/sBws8MSXN7A) made by [Brad Traversy](https://github.com/bradtraversy).

## Usage

```
# Install dependencies
npm install
```

```
# Run app
npm start
```

```
# Build app
npm run build
```
